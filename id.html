IDs e JavaScript

Até agora só usamos os comandos 'alert' e 'prompt' para interagir com nosso site. Mas e se eu quiser mudar o conteúdo dentro do nosso site?

Para isso servem os IDs. Eles são nomes que vocês vão dar a pedaços do site. Para fazer isso, basta colocar a propriedade 'id' dentro da tag que vocês quer dar um nome

Depois que tiver feito isso, você vai fazer uma referência no arquivo JavaScript para poder mexer com o conteúdo.

Vamos apenas mudar o texto desta página aqui. Você vai poder fazer suas próprias alterações em páginas dos exercícios.

O programa do JavaScript está programado para rodar quando apertamos o botão do lado (vide Botões). Você vai aprender a usá-los na seção de botões deste artigo.

Acesse aqui o arquivo JavaScript desta página para explicações de como funciona.
