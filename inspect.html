Como inspecionar no computador

Inspecionar elemento é um recurso dos navegadores da web para poder olhar para os itens do site e seu código HTML relacionado.

Para poder fazer isso, basta clicar com o botão direito do mouse no elemento que você quer olhar e selecionar 'inspecionar elemento'.

Você pode também ver todo o código HTML do site presionando ctrl+shift+i e depois você pode navegar pelo código para achar seu elemente.

Ele mostra qual elemento está sendo selecionado na página. Você também pode alterar o código HTML da página nessa aba, mas as modificações não serão permanentes.

Atenção: Isso não funciona no Internet Explorer nem no Microsoft Edge

Como inspecionar no celular

Inspecionar no celular é um pouco diferente. Dá para ver e editar todo o HTML do site, mas não qual elemento é qual. Ainda assim, tente olhar e ver de qual

elemento estamos falando. Par acessar o HTML do site, clique no URL, clique no lápis para editar o URL e coloque 'view-source:' antes de tudo que esstiver escrito.
