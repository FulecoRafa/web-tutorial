Formulário é uma série de campos onde preenchemos dados que vão ajudar o usuário a interagir com nosso site.

Para criar esses campos, vamos usar a tag 'form' e dentro dela, os itens com a tag 'input'

Cada input pode ser de um tipo, definido na propriedade 'type'. No formulário abaixo, inspecione os elementos e veja quais os tipos usados.

Cada input tem também um 'name', que funciona de forma similar ao 'id'. Por fim, cada input tem um 'value', que é exatamente o que está escrito dentro da caixinha
